Windows下为go程序加入图标资源

Windows下为go程序加入图标资源 1.准备图标资源如demo.ico，适用Windows的图标 2.新建rc文件，如demo.rc 内容如下 IDI_ICON1 ICON "daily.ico" 3.使用windres编译demo.rc。（需预先安装TDM-GCC-64） windres -o daily.syso daily.rc 64位环境下编译出32位的链接文件 需要额外参数 windres --output-format=coff --target=pe-i386 -o daily.syso daily.rc 4.使用go工具编译 go build 或 go build -ldflags="-H windowsgui" 5.如果是多平台开发，非Windows平台不需要rc资源，可以建立以下批处理 build.bat @echo offwindres -o daily.syso daily.rcgo builddel daily.syso
